### 网易云音乐介绍
```
网易云音乐是一款由网易开发的音乐产品，是网易杭州研究院的成果[1]，依托专业音乐人、DJ、好友推荐及社交功能，在线音乐服务主打歌单、社交、大牌推荐和音乐指纹，以歌单、DJ节目、社交、地理位置为核心要素，主打发现和分享。[2]
该产品2013年4月23日正式发布，截止2017年04月，产品已经包括iPhone、Android、Web、PC、iPad、WP8、Mac、Win10UWP、Linux九大平台客户端[3][4][5]。2015年1月16日，网易云音乐荣膺百度中国好应用“年度优秀视觉设计奖”。
2018年5月30日，屈臣氏联合网易云音乐打造的音乐主题门店在广州中华广场开业，区别于普通门店，店内设置了电音化妆间，消费者可现场体验“音乐主题”妆容。[6]
```

### 网易云音乐使用指南
```
1.用户可以根据自己的喜好创建、收藏、分享歌单，应用以用户创造的歌单为基本线索。

2.用户可以导入原有手机通讯录、 SNS关系，或者借助网易云音乐自身 LBS功能构建音乐社交圈。

3.用户可以建立自己的主页，可以互相关注，分享音乐，用户的动态在这个圈子里即时呈现。

4.网易云音乐包含DJ及音乐人独家自制节目。

5.应用音乐指纹技术，通过听取音乐旋律，帮助用户找到歌曲。
```

### 网易云音乐推出“云梯计划” 扶持内容创作者 构建共赢商业生态
```
5月18日，网易云音乐CEO朱一闻在「音乐原聚力」营销盛典现场演讲时表示，网易云音乐将推出“云梯计划”，扶持包括音乐人、视频作者在内的原创内容创作者。目前，“云梯计划”已正式启动。网易云音乐宣布，平台将通过广告分成、原创内容激励、自助数字专辑售卖、音乐人资源推广等一系列产品功能的创新改进和运营机制的制定，扶持广大内容创作者，推动其与平台共享数字音乐产业发展成果。
据介绍，目前，网易云音乐“云梯计划”将从以下四方面对内容创作者进行大力扶持：
一、音乐人广告分成：正式上线音乐人评论区广告分成功能。独立音乐人可在网易云音乐平台开通歌曲评论区广告分成功能，其后，音乐人可从相关歌曲的评论区广告中获得最高80%的收益分成。
二、短视频现金激励：升级短视频现金激励计划。今年3月，网易云音乐推出短视频现金激励计划。此次升级，网易云音乐将在原有基础上，再从新上线的视频页面信息流广告中拿出30%收益，共同构成激励基金，为更多音乐相关原创优质视频作者提供更丰富激励。优质视频创作者也可获得相应流量包，助力原创视频传播推广。
三、自助数字专辑售卖：开放自助数字专辑售卖功能。网易云音乐平台音乐人指数达450分的独立音乐人，均可在线上开通自助数字专辑售卖功能。
四、推出虚拟货币“云豆”：助力独立音乐人获取更多曝光资源。通过这一业内首创的虚拟货币体系，独立音乐人可使用云豆在网易云音乐平台按需兑换推广流量包、定向推送私信、动态粉丝头条等多种推广资源。未来可兑换资源将会更加丰富。
 通过“云梯计划”中多途径内容变现与多渠道流量推广等扶持内容，网易云音乐希望成为助推更多内容创作者上升的阶梯，帮助他们取得更好发展。据了解，音乐人评论区广告分成功能3月底开始试运行，网易音乐人烟把儿开通这一功能后，半个月时间即获得超5万元的评论区广告分成收入。

```
